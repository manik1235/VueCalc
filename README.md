# vue-calc

## Clone
```
git clone git@github.com:manik1235/VueCalc.git
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```
or
```
npm run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
